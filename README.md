# Backend API Documentations

This repository contains comprehensive API documentation for various backend projects. Each project has its own dedicated documentation file in Markdown format, making it easy to share and reference.

## Repository Structure

- `README.md` - This file, providing an overview of the repository
- Project documentation files (`.md`) - Individual API docs for each project

## Available Documentations

- [HeroWall API](HEROWALL_API.md) - Comprehensive API documentation for HeroWall, a professional digital collectibles platform built with NestJS. Includes authentication, card management, and collection features.

## How to Use

1. Navigate to the specific project's documentation file
2. Read through the API endpoints, request/response formats, and examples
3. Use the documentation to integrate with the respective APIs

## Contributing

To add documentation for a new project:

1. Create a new Markdown file with the project name (e.g., `project-name-api.md`)
2. Follow the standard documentation format:
   - Project overview
   - Authentication
   - API endpoints with examples
   - Error handling
   - Changelog
